# bilibili-quickdo
b站H5播放器快速操作

## 功能
- 双击全屏
- ```+``` ```-```键调节播放速度
- ```F```键全屏
- ```W```键宽屏
- ```K```键暂停/播放
- ```J``` ```L```键前进/后退5s
- ```D```键开启/关闭弹幕
- ```N``` ```M```调节音量(不记忆)
- 播放器右侧设置-高级选项 可以设置自动播放、自动宽屏、超级宽屏、双击全屏和关闭弹幕
