# bilibili-quickdo
b站H5播放器快速操作

## 功能
- 双击全屏
- ```+``` ```-```键调节播放速度
- ```F```键全屏
- ```W```键宽屏
- ```P```键暂停/播放
- ```D```键开启/关闭弹幕
- ```K``` ```L```切换前/后一分P
- 回车键快速进入弹幕编辑状态，全屏模式下也可以
- 播放器右侧设置-高级选项 可以设置自动播放、宽屏和关闭弹幕
